Terraform is an Infrastructure as Code (IaC) tool for building, changing, and versioning infrastructure safely and efficiently.

## Overview

Learn how to manage infrastructure declaratively using Terraform across multiple cloud providers.

## What you'll learn

- Terraform core concepts and workflow
- HCL (HashiCorp Configuration Language) syntax
- Providers and resources
- State management
- Modules and reusability
